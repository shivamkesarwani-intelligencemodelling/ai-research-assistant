# AI Research Assistant

An AI-powered research assistant that searches the web, gathers information from multiple sources, and generates concise research reports with citations.

## Features

* Web search
* Source extraction
* AI-generated summaries
* Research reports
* Citation support

## Tech Stack

* Python
* OpenAI API
* BeautifulSoup
* Requests

1. Clone the repository.
2. Create a virtual environment.
3. Install dependencies.
